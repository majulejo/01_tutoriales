Text alignment/Alineación del texto

Realinee fácilmente el texto con los componentes mediante clases de alineación de texto. Para la alineación inicial, final y centrada, existen clases adaptables que utilizan los mismos puntos de corte de ancho de ventana gráfica que el sistema de cuadrícula.

BOOTSTRAP EJEMPLOS alinear texto

text-center Texto alineado al centro

text-start texto alineado a la izquierda o start en todos los tamaños del viewport

text-center Texto alineado al centro en todos los tamaños del viewport

text-end texto alineado a la derecha en todos los tamaños del viewport

SM Texto alineado final en ventanas gráficas de tamaño pequeño (SM) o más anchas.

MD Texto alineado final en ventanas gráficas de tamaño MD (mediano) o más ancho.

LG Texto alineado final en ventanas gráficas de tamaño LG (grande) o más ancho.

XL Texto alineado final en ventanas gráficas de tamaño XL (extra grande) o más ancho.

XXL Texto alineado final en ventanas gráficas de tamaño XXL (extra extra grande) o más ancho.

Tenga en cuenta que no ofrecemos clases de utilidad para texto justificado. Si bien el texto justificado puede ser más atractivo estéticamente, hace que el espaciado entre palabras sea más aleatorio y, por lo tanto, más difícil de leer.

Sass utilities API/API de utilidades de Sass

Las utilidades de fuentes y texto se declaran en nuestra API de utilidades en [Nombre del archivo] scss/_utilities.scss. Aprenda a usar la API de utilidades.

scss/_utilitiess.scss 
            "font-family": (
    property: font-family,
    class: font,
    values: (monospace: var(--#{$prefix}font-monospace))
),
"font-size": (
    rfs: true,
    property: font-size,
    class: fs,
    values: $font-sizes
),
"font-style": (
    property: font-style,
    class: fst,
    values: italic normal
),
"font-weight": (
    property: font-weight,
    class: fw,
        values: (
    lighter: $font-weight-lighter,
    light: $font-weight-light,
    normal: $font-weight-normal,
    medium: $font-weight-medium,
    semibold: $font-weight-semibold,
    bold: $font-weight-bold,
    bolder: $font-weight-bolder
    )
),
"line-height": (
    property: line-height,
    class: lh,
    values: (
    1: 1,
    sm: $line-height-sm,
    base: $line-height-base,
    lg: $line-height-lg,
    )
),
"text-align": (
    responsive: true,
    property: text-align,
    class: text,
    values: (
    start: left,
    end: right,
    center: center,
    )
),
"text-decoration": (
    property: text-decoration,
    values: none underline line-through
),
"text-transform": (
    property: text-transform,
    class: text,
    values: lowercase uppercase capitalize
),
"white-space": (
    property: white-space,
    class: text,
    values: (
    wrap: normal,
    nowrap: nowrap,
    )
),
"word-wrap": (
    property: word-wrap word-break,
    class: text,
    values: (break: break-word),
    rtl: false
),
