Las tarjetas se crean con la menor cantidad posible de marcado y estilos, pero aun así ofrecen un gran control y personalización. Creadas con flexbox, ofrecen una alineación sencilla y se integran a la perfección con otros componentes de Bootstrap. No tienen espaciado marginpredeterminado, así que usa las utilidades de espaciado según sea necesario.
