TTDFEC-1

Cards

Example
Las tarjetas se crean con la menor cantidad posible de marcado y estilos, pero aun así ofrecen un gran control y personalización. Creadas con flexbox, ofrecen una alineación sencilla y se integran a la perfección con otros componentes de Bootstrap. No tienen espaciado margin predeterminado, así que usa las utilidades de espaciado según sea necesario.
A continuación se muestra un ejemplo de una tarjeta básica con contenido mixto y un ancho fijo. Las cartas no tienen anchura fija para empezar, por lo que llenarán naturalmente la anchura completa de su elemento padre. Esto se personaliza fácilmente con nuestras diversas opciones de tallas.

TTDFEC-5 TAMAÑOS

Sizing

Las tarjetas a las que no se le aplica un "width", tendrán un ancho del 100 % a menos que se indique lo contrario.
Puedes modificar esto según sea necesario con CSS personalizado, clases de cuadrícula, mixins de Sass para cuadrícula o utilidades.
Uso del marcado de cuadrícula
Usando la cuadrícula, envuelva las tarjetas en columnas y filas según sea necesario.
col-sm-6

Esta clase define el ancho de la columna en dispositivos con un tamaño de pantalla "small" (pequeño) o superior.
En dispositivos con un ancho de pantalla ≥ 576px (pantallas pequeñas o mayores), este div ocupará 6 columnas de un total de 12 columnas disponibles en la cuadrícula de Bootstrap. En pantallas más pequeñas (por ejemplo, móviles con un ancho < 576px), el div se comportará como un bloque completo (width: 100%) porque no tiene una clase específica para pantallas extra pequeñas (col-xxs-*).

<div class="row g-0">

1. Clase row

Descripción :

La clase row es una de las clases fundamentales de Bootstrap. Define un contenedor que organiza elementos en una fila horizontal.

Funcionalidad :

Actúa como un contenedor flexible (flexbox) para alinear elementos hijos (generalmente columnas).
Los elementos dentro de una fila (row) deben ser columnas (col-*) para que se distribuyan correctamente.
Por defecto, los elementos dentro de una fila tienen un margen negativo (margin-left: -15px; margin-right: -15px;) para compensar el padding de las columnas.
2. Clase g-0

Descripción :

La clase g-0 es una utilidad de Bootstrap que controla el esparcimiento (gutter) entre las columnas dentro de una fila.

Funcionalidad :

El "gutter" es el espacio horizontal y vertical entre las columnas. g-0 elimina completamente este espacio, haciendo que las columnas se ajusten sin márgenes entre ellas.
Esto es útil cuando quieres un diseño más compacto o cuando necesitas eliminar los espacios predeterminados entre columnas.
Comportamiento Combinado
Cuando usas <div class="row g-0">, estás creando una fila en la que:
Las columnas dentro de esta fila no tendrán ningún espacio (gutter) entre ellas.
Los elementos hijos (columnas) se distribuirán horizontalmente utilizando el sistema de cuadrícula de Bootstrap.

Grid cards/Tarjetas de cuadrícula
Usa el sistema de cuadrícula de Bootstrap y sus clases .row-cols para controlar cuántas columnas de cuadrícula (que rodean tus tarjetas) se muestran por fila. Por ejemplo, .row-cols-1 distribuye las tarjetas en una columna, y .row-cols-md-2 divide cuatro tarjetas con el mismo ancho en varias filas, desde el punto de interrupción medio hacia arriba.
<div class="row row-cols-1 row-cols-md-2 g-4">

1. Estructura del div Este div es un contenedor que utiliza clases de Bootstrap , un framework de diseño web responsive y popular. Bootstrap permite crear interfaces web adaptables y responsivas sin necesidad de escribir CSS personalizado en muchos casos.

2. Clases aplicadas al div

a) row La clase row es una de las clases fundamentales de Bootstrap. Su propósito es definir un contenedor que organiza su contenido en una disposición de tipo "fila". Los elementos dentro de esta fila (generalmente columnas) se distribuyen horizontalmente de forma predeterminada. También incluye estilos para manejar márgenes negativos (margin-left y margin-right) que compensan los paddings de las columnas internas.
b) row-cols-1 Esta clase define cuántas columnas debe haber por defecto dentro de la fila. En este caso, row-cols-1 indica que, por defecto, solo habrá una columna en dispositivos pequeños o cuando no se apliquen reglas específicas para otros tamaños de pantalla.

c) row-cols-md-2

Esta clase modifica el comportamiento de la fila para dispositivos con un ancho de pantalla mediano (md: medium) o superior. El sufijo -md indica que esta regla se aplicará a pantallas con un ancho mínimo de 768px (según el sistema de breakpoints de Bootstrap). row-cols-md-2 establece que, en estos dispositivos, habrá dos columnas en lugar de una.

d) g-4 La clase g-4 controla el espaciado entre las columnas (también conocido como "gutters"). El valor 4 indica que el espacio entre las columnas será igual a 16px (basado en la escala de espaciado de Bootstrap, donde 1 = 4px, 2 = 8px, etc.). Este espaciado se aplica tanto horizontal como verticalmente entre las columnas.

3. Comportamiento esperado

Con base en las clases aplicadas, este div tendrá el siguiente comportamiento:
• En pantallas pequeñas (xs):
 • Habrá una sola columna (row-cols-1), lo que significa que todos los elementos dentro de esta fila se mostrarán apilados verticalmente.
 • Habrá un espaciado de 16px (g-4) entre los elementos.
• En pantallas medianas (md) o más grandes:
 • Se mostrarán dos columnas (row-cols-md-2), distribuyendo los elementos horizontalmente.
 •El espaciado entre las columnas seguirá siendo de 16px (g-4).
4. Posibles usos Este tipo de estructura es común en diseños responsivos, como:
 • Tarjetas : Mostrar tarjetas de productos o servicios que se ajustan automáticamente según el tamaño de la pantalla.
 • Galerías de imágenes : Crear una galería que se adapte a diferentes dispositivos.
 • Formularios : Organizar campos de formulario en columnas para mejorar la experiencia del usuario.
