Utilice las utilidades de ajuste de objetos para modificar la forma en que se debe redimensionar el contenido de un elemento reemplazado , como un <img> o , para que se ajuste a su contenedor <video>.

How it works/Cómo funciona

Cambie el valor de la propiedad object-fit con nuestras clases de utilidad adaptables object-fit . Esta propiedad indica al contenido que llene el contenedor principal de diversas maneras, como conservando la relación de aspecto o expandiéndolo para ocupar el máximo espacio posible. .

Las clases para el valor de object-fit se nombran con el formato .object-fit-{value}. Elija entre los siguientes valores:

  • contain
  • cover
  • fill
  • scale (para reducir la escala)
  • none

Examples/ Ejemplos

Añade la clase .object-fit-{value} al elemento reemplazado :

object-fit: contain;

object-fit: cover;

object-fit: fill;

object-fit: scale-down;

object-fit: none;

HTML
<img src="..." class="object-fit-contain border rounded" alt="...">
<img src="..." class="object-fit-cover border rounded" alt="...">
<img src="..." class="object-fit-fill border rounded" alt="...">
<img src="..." class="object-fit-scale border rounded" alt="...">
<img src="..." class="object-fit-none border rounded" alt="...">

Responsive/ Sensible

También existen variaciones de respuesta para cada object-fitvalor que utilice el formato .object-fit-{breakpoint}-{value}, para las siguientes abreviaturas de puntos de interrupción: sm, md, lg, xly xxl. Las clases se pueden combinar para obtener diversos efectos según sea necesario.

object-fit-sm-contain

object-fit-md-contain

object-fit-lg-contain

object-fit-xl-contain

object-fit-xxl-contain

HTML
<img src="..." class="object-fit-sm-contain border rounded" alt="...">
<img src="..." class="object-fit-md-contain border rounded" alt="...">
<img src="..." class="object-fit-lg-contain border rounded" alt="...">
<img src="..." class="object-fit-xl-contain border rounded" alt="...">
<img src="..." class="object-fit-xxl-contain border rounded" alt="...">

Video/ Video

Las utilidades responsivas .object-fit-{value} y .object-fit-{breakpoint}-{value} también funcionan con elementos <video > . para ajustarse automáticamente al valor actual del elemento .

Sass utilities API/API de utilidades de Sass

Las utilidades de interacción se declaran en nuestra API de utilidades en [nombre del archivo] scss/_utilities.scss. Aprenda a usar la API de utilidades. incluidas (también conocidas como variables CSS) para colores y más.

scss/_utilities.scss 
"link-opacity": (
    css-var: true,
    class: link-opacity,
    state: hover,
    values: (
      10: .1,
      25: .25,
      50: .5,
      75: .75,
      100: 1
    )
  ),
  "link-offset": (
    property: text-underline-offset,
    class: link-offset,
    state: hover,
    values: (
      1: .125em,
      2: .25em,
      3: .375em,
    )
  ),
  "link-underline": (
    property: text-decoration-color,
    class: link-underline,
    local-vars: (
      "link-underline-opacity": 1
    ),
    values: map-merge(
      $utilities-links-underline,(
        null: rgba(var(--#{$prefix}link-color-rgb), var(--#{$prefix}link-underline-opacity, 1)),
      )
    )
  ),
  "link-underline-opacity": (
    css-var: true,
    class: link-underline-opacity,
    state: hover,
    values: (
      0: 0,
      10: .1,
      25: .25,
      50: .5,
      75: .75,
      100: 1
    ),
  ),
