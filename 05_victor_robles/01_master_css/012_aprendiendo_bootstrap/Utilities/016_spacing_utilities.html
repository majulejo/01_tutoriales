Margin and padding/Margen y relleno

Asigna valores adaptables margin o padding a un elemento o a un subconjunto de sus lados con clases abreviadas. Incluye compatibilidad con propiedades individuales, todas las propiedades y propiedades verticales y horizontales. Las clases se construyen a partir de un mapa Sass predeterminado que va desde .25rem hasta 3rem .

¿Usas el módulo de diseño CSS Grid? Considera usar la utilidad gap .

Notation /Notación

Las utilidades de espaciado que se aplican a todos los puntos de interrupción, desde xs hasta xxl, no incluyen abreviatura. Esto se debe a que estas clases se aplican desde min-width: 0 hacia arriba, por lo que no están vinculadas por una consulta de medios. Sin embargo, los puntos de interrupción restantes sí incluyen una abreviatura.

Las clases se nombran utilizando el formato {property}{sides}-{size} para xs y {property}{sides}-{breakpoint}-{size} para sm, md, lg, xl, y xxl.

Cuando la propiedad es una de las siguientes:

  • m- para clases que establecen margin.
  • p- para clases que establecen padding.

Donde lados es uno de:

  • t- para clases que establecen margin-top o padding-top.
  • b- para clases que establecen margin-bottom o padding-bottom.
  • s-(inicio) para clases que establecen margin-left or padding-left en LTR,o margin-right or padding-right en RTL .
  • e-(final) para clases que establecen margin-right or padding-right en LTR, o margin-left or padding-left en RTL .
  • x- para clases que establecen tanto *-left como *-right .
  • y- para clases que establecen tanto *-top como *-bottom.
  • en blanco- para clases que establecen un margin o padding en los cuatro lados del elemento.

Donde el tamaño es uno de los siguientes:

  • 0- para clases que eliminan el margin o padding configurándolo en0
  • 1- (por defecto) para clases que establecen el margin o padding en$spacer * .25
  • 2- (por defecto) para clases que establecen el margin o padding en$spacer * .5
  • 3- (por defecto) para clases que establecen el margin o padding en$spacer
  • 4- (por defecto) para clases que establecen el margin o padding en$spacer * 1.5
  • 5- (por defecto) para clases que establecen el margin o padding en$spacer * 3
  • auto- para clases que lo configuran margin en automático

(Puede agregar más tamaños agregando entradas a la variable $spacers del mapa Sass).

Horizontal centering/Centrado horizontal

Además, Bootstrap también incluye una clase .mx-auto para centrar horizontalmente el contenido de nivel de bloque de ancho fijo (es decir, contenido que tiene display: block y un width conjunto) estableciendo los márgenes horizontales en auto.

Centered element
HTML 
<div class="mx-auto p-2 border rounded bg-warning-subtle" style="width: 200px;">
    Centered element
</div>

Negative margin/Margen negativo

En CSS, las propiedades margin pueden usar valores negativos ( padding no se pueden). Estos márgenes negativos están deshabilitados por defecto , pero se pueden habilitar en Sass configurando [Nombre del elemento] $enable-negative-margins: true .

Gap/Brecha

Al usar display: grid o display: flex, puede usar de la utilidades de gap en el elemento padre. Esto le ahorra tener que agregar utilidades de margen a elementos secundarios individuales de una cuadrícula o contenedor flexible. Las utilidades de Gap son adaptables por defecto y se generan a través de nuestra API de utilidades, basada en el $spacers mapa de Sass.

CSS

Sass maps/Mapas de Sass

Las utilidades de espaciado se declaran a través del mapa Sass y luego se generan con nuestra API de utilidades.

scss/_variables.scss 
    $spacer: 1rem; 
    $spacer:(
        0:0,
    1:$spacer*.25,
    2:$spacer*.5,
    3:$spacer,
    4:$spacer*1.5,
    5:$spacer*3,    
    );
Sass utilities API/API de utilidades de Sass
