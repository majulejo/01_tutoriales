Z-index

Utilice nuestras utilidades de bajo nivel z-index para cambiar rápidamente el nivel de pila de un elemento o componente.

Example/Ejemplo

Usa utilidades z-index para apilar elementos. Requiere un valor position distinto de static , que se puede configurar con estilos personalizados o con nuestras utilidades de posición

Estas utilidades z-index se denominan de "bajo nivel" debido a sus valores predeterminados de " -1" hasta 3 , que utilizamos para el diseño de componentes superpuestos. Los valores z-index de alto nivel se utilizan para componentes superpuestos como ventanas modales y descripciones emergentes.

z-3
z-2
z-1
z-0
z-n1
HTML
    <!-- Caja z-3 -->
    <div class="z-3 position-absolute p-5 rounded-3 border border-dark"
        style="width:75px; height:75px; background-color:#c1d663; top: 10px; left: 10px;">z-3
    </div>

    <!-- Caja z-2 -->
    <div class="z-2 position-absolute p-5 rounded-3 border border-dark"
        style="width:75px; height:75px; background-color:#6fdae8; top: 50px; left: 50px;">
        <span>z-2</span>
    </div>

    <!-- Caja z-1 -->
    <div class="z-1 position-absolute p-5 rounded-3 border border-dark"    style="width:75px; height:75px; background-color:#c1d663; top: 90px; left: 90px;">
         <span>z-1</span>
     </div>

     <!-- Caja z-0 -->
     <div class="z-0 position-absolute p-5 rounded-3 border border-dark"
          style="width:75px; height:75px; background-color:#6fdae8; top: 130px; left: 130px;">
         <span>z-0</span>
     </div>

     <!-- Caja z-n1 -->
     <div class="z-n1 position-absolute p-5 rounded-3 border border-dark"
          style="width:75px; height:75px; background-color:#c1d663; top: 170px; left: 170px;">
         <span>z-n1</span>
     </div>

Overlays /Superposiciones

Los componentes de superposición de Bootstrap (menú desplegable, modal, offcanvas, popover, toast y tooltip) tienen sus propios z-indexvalores para garantizar una experiencia utilizable con “capas” competitivas de una interfaz.

Lea sobre ellos en la página de diseño z-index.

Component approach/Enfoque de componentes

En algunos componentes, utilizamos nuestros valores z-index de bajo nivel para administrar elementos repetidos que se superponen entre sí (como botones en un grupo de botones o elementos en un grupo de listas).

Conozca nuestro enfoque z-index.

CSS

Sass Maps/Mapas de Sass

Personalice este mapa Sass para cambiar los valores disponibles y las utilidades generadas.

scss/_variables.scss 
            $zindex-levels: (
    n1: -1,
     0: 0,
    1:1,
    2:2,
    3:3
    ),
Sass utilities API/API de utilidades de Sass

Las utilidades de fuentes y texto se declaran en nuestra API de utilidades en [Nombre del archivo] scss/_utilities.scss. Aprenda a usar la API de utilidades.

scss/_utilities.scss 
                    "z-index": (
                        property: z-index,
                            class: z,
                            values: $zindex-levels,
              )
