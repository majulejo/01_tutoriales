Enable flex behaviors / Habilitar comportamientos flexibles

Aplique utilidades display para crear un contenedor flexbox y transformar elementos secundarios directos en elementos flex. Los contenedores y elementos flex se pueden modificar aún más con propiedades flex adicionales.

¡Soy un contenedor flexible!

HTML

<div class="d-flex border rounded " style="background-color: #f0a4f0">¡Soy un contenedor flexible!</div>
¡Soy un contenedor flexbox en línea!

HTML

<div class="d-flex border rounded " style="background-color: #f0a4f0">¡Soy un contenedor flexbox en línea!</div>

Direction / Dirección

Establezca la dirección de los elementos flexibles en un contenedor flexible con las utilidades de dirección. En la mayoría de los casos, puede omitir la clase horizontal, ya que el valor predeterminado del navegador es row. Sin embargo, puede encontrar situaciones en las que necesite establecer este valor explícitamente (como en diseños adaptables). Úselo .flex-rowpara establecer una dirección horizontal (la predeterminada del navegador) o .flex-row-reversepara iniciar la dirección horizontal desde el lado opuesto.
