sizing/Apresto

Haga fácilmente un elemento tan ancho o tan alto con nuestras utilidades de ancho y alto.

En esta página:

  • Relative to the parent/Relativo al padre
  • Relative to the viewport/Relativo a la ventana gráfica

  • Css

    • Sass utilities API/API de utilidades de Sass

Relative to the parent/Relativo al padre

Las utilidades de ancho y alto se generan desde la API de utilidades en _utilities.scss. Incluye compatibilidad con 25% , 50% , 75% , 100% y auto de forma predeterminada. Modifique estos valores según sea necesario para generar diferentes utilidades aquí.

Width 25%
Width 50%
Width 75%
Width 100%
Width auto
HTML
 
<div class="w-25 p-3 border rounded" style="width:25%; background-color: #F5F5DC;">Width 25%</div>
<div class="w-50 p-3 border rounded" style="width:50%; background-color: #F5F5DC;">Width 50%</div>
<div class="w-75 p-3 border rounded" style="width:75%; background-color: #F5F5DC;">Width 75%</div>
<div class="w-100 p-3 border rounded" style="width:100%; background-color: #F5F5DC;">Width 100%</div>
<div class="w-auto p-3 border rounded" style="width:auto; background-color: #F5F5DC;">Width auto</div>
Height 25%
Height 50%
Height 75%
Height 100%
Height auto
HTML
         
<div style="height: 100px;">
    <div class="h-25 d-inline-block border rounded" style="width: 120px;background-color: #8FBC8F">Height 25%</div>
    <div class="h-50 d-inline-block border rounded" style="width: 120px; background-color: #8FBC8F">Height 50%</div>
    <div class="h-75 d-inline-block border rounded" style="width: 120px; background-color: #8FBC8F">Height 75%</div>
    <div class="h-100 d-inline-block border rounded" style="width: 120px; background-color: #8FBC8F">Height 100%</div>
    <div class="h-auto d-inline-block border rounded" style="width: 120px; background-color: #8FBC8F">Height auto</div>
</div>

También puedes utilizar las utilidades max-width: 100%; o max-height: 100%; según sea necesario.

Max-width 100%
HTML
<div style="width: 50%; height: 100px;">
    <div class="mw-100" style="width: 200%;">Max-width 100%</div>
</div>
Max-height 100%
HTML
                         
<div class="border rounded" style="height: 100px; width:auto; background-color: #f8c621;">
    <class="mh-100 border rounded" style="width: 100px; height: 200px;"">Max-height 100%</div>
</div>

Relative to the viewport /Relativo a la ventana gráfica

También puedes utilizar utilidades para establecer el ancho y la altura en relación con la ventana gráfica.

Min-width 100vw
Min-height 100vh
Width 100vw
Height 100vh

CSS

Sass utilities API/API de utilidades de Sass

Las utilidades de sombra se declaran en nuestra API de utilidades en [nombre del archivo] scss/_utilities.scss. Aprenda a usar la API de utilidades.

scss/_utilities.scss 
"width": (
property: width,
    class: w,
    values: (
        25: 25%,
        50: 50%,
        75: 75%,
        100: 100%,
        auto: auto
    )
  ),
"max-width": (
  property: max-width,
  class: mw,
  values: (100: 100%)
),
"viewport-width": (
  property: width,
  class: vw,
  values: (100: 100vw)
),
"min-viewport-width": (
  property: min-width,
  class: min-vw,
  values: (100: 100vw)
),
"height": (
  property: height,
  class: h,
  values: (
    25: 25%,
    50: 50%,
    75: 75%,
    100: 100%,
    auto: auto
  )
),
"max-height": (
  property: max-height,
  class: mh,
  values: (100: 100%)
),
"viewport-height": (
  property: height,
  class: vh,
  values: (100: 100vh)
),
"min-viewport-height": (
  property: min-height,
  class: min-vh,
  values: (100: 100vh)
),
