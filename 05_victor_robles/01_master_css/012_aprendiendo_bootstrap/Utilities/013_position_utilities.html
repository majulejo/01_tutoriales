Arrange elements/Organizar elementos

Organiza elementos fácilmente con las utilidades de posicionamiento de bordes. El formato es {property}-{position} .

Cuando la propiedad es una de las siguientes:

  • top- para la posición vertical top
  • start- para la posición horizontal (en LTR) left
  • bottom- para la posición vertical bottom
  • end- para la posición horizontal (en LTR) right

Donde posición es una de:

  • 0- para la posición del borde 0
  • 50- para la posición del borde 50%
  • 100- para la posición del borde 100%

(Puede agregar más valores de posición agregando entradas a la variable del mapa Sass $position-values).

HTML 
<div class="position-relative">
    <div class="bg-warning-subtle  border rounded  "style="height: 250px;">
        <div class="position-absolute top-0 start-0 rounded" style="width:50px;height:50px;background-color:rgb(119, 128, 130);"></div>
        <div class="position-absolute top-0 end-0 rounded" style="width:50px;height:50px;background-color:lightblue;"></div>
        <div class="position-absolute top-50 start-50 rounded" style="width:50px;height:50px;background-color:lightblue;"></div>
        <div class="position-absolute bottom-50 end-50 rounded" style="width:50px;height:50px;background-color:lightblue;"></div>
        <div class="position-absolute bottom-0 start-0 rounded" style="width:50px;height:50px;background-color:lightblue;"></div>
        <div class="position-absolute bottom-0 end-0 rounded" style="width:50px;height:50px;background-color:lightblue;"></div>
    </div>
</div>

Center elements/Elementos centrales

Además, también puedes centrar los elementos con la clase de utilidad de transformación .translate-middle.

Esta clase aplica las transformaciones translateX(-50%) y translateY(-50%) al elemento que, en combinación con las utilidades de posicionamiento de bordes, permite centrar absolutamente un elemento.overflow-X para afectar el desbordamiento de contenido verticalmente.

HTML 
<div class="position-relative">
    <div class="bg-warning-subtle  border rounded  "style="height: 150px;">
        <div class="position-absolute top-0 start-0 translate-middle rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute top-0 start-50 translate-middle rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute top-0 start-100 translate-middle rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute top-50 start-0 translate-middle rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute top-50 start-50 translate-middle rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute top-50 start-100 translate-middle rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute top-100 start-0 translate-middle rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute top-100 start-50 translate-middle rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute top-100 start-100 translate-middle rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
    </div>
</div>

Al agregar clases .translate-middle-x (-50%) o .translate-middle-y , los elementos se pueden posicionar solo en dirección horizontal o vertical.

HTML 
<div class="position-relative">
    <div class="bg-warning-subtle  border rounded  "style="height: 150px;">
        <div class="position-absolute top-0 start-0 rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute top-0 start-50 translate-middle-x rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute top-0 end-0 rounded" style="width:25px;height:25px;background-color:#085425;"></div>
        <div class="position-absolute top-50 start-0 translate-middle-y rounded" style="width:25px;height:25px;background-color:#085425;"></div>
        <div class="position-absolute top-50 start-50 translate-middle rounded" style="width:25px;height:25px;background-color:#73d914;"></div>
        <div class="position-absolute top-50 end-0 translate-middle-y rounded" style="width:25px;height:25px;background-color:#73d914;"></div>
        <div class="position-absolute bottom-0 start-0 rounded" style="width:25px;height:25px;background-color:#119d8c;"></div>
        <div class="position-absolute bottom-0 start-50 translate-middle-x rounded" style="width:25px;height:25px;background-color:#119d8c;"></div>
        <div class="position-absolute bottom-0 end-0 rounded" style="width:25px;height:25px;background-color:#07bbdf;"></div>
        <div class="position-absolute bottom-0 start-25 translate-middle-x rounded" style="width:25px;height:25px;background-color:#f75205;"></div>
    </div>
</div>

Puedes usar estas clases con componentes existentes para crear nuevos. Recuerda que puedes ampliar su funcionalidad añadiendo entradas a la variable $position-values .

HTML
<div class="position-relative m-4">
    <div class="progress" role="progressbar" aria-label="Progress" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100" style="height: 1px;">
      <div class="progress-bar" style="width: 50%"></div>
    </div>
    <button type="button" class="position-absolute top-0 start-0 translate-middle btn btn-sm btn-primary rounded-pill" style="width: 2rem; height:2rem;">1</button>
    <button type="button" class="position-absolute top-0 start-50 translate-middle btn btn-sm btn-primary rounded-pill" style="width: 2rem; height:2rem;">2</button>
    <button type="button" class="position-absolute top-0 start-100 translate-middle btn btn-sm btn-secondary rounded-pill" style="width: 2rem; height:2rem;">3</button>
</div>
