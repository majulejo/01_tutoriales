5. Resumen de opciones

Tipo de estilo Cómo aplicarlo
Clases de Bootstrap Usa .link-*, .text-*, .fw-*, .fst-*, etc.
CSS personalizado Define tus propias reglas en una etiqueta <style> o archivo CSS externo.
Combinación Combina clases de Bootstrap con CSS personalizado para un diseño más flexible.
Pseudo-clases Usa :hover, :active, :focus, y :visited para estilos en diferentes estados.
Transiciones Agrega transition: all 0.3s ease; para animaciones suaves.
