Tabla de referencia para link-offset-*

Clase Descripción Valor aproximado de text-underline-offset
link-offset-1 Desplazamiento mínimo del subrayado. 0.125em
link-offset-2 Desplazamiento mediano del subrayado. 0.25em
link-offset-3 Desplazamiento mayor del subrayado. 0.375em
link-offset-4 No existe en Bootstrap 5. No aplicable
Custom CSS Desplazamiento personalizado usando CSS. Depende del valor definido (ejemplo: 0.625em)

Explicación:

  • Las clases link-offset-1, link-offset-2, y link-offset-3 son parte de Bootstrap 5.
  • No hay soporte para link-offset-4 o valores mayores en Bootstrap 5.
  • Para valores mayores, puedes usar CSS personalizado con la propiedad text-underline-offset.
