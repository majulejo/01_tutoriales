Shadows/Sombras

Agregue o elimine sombras a los elementos con las utilidades box-shadow.

En esta página:

  • Examples/Ejemplos

  • Css

    • Sass variables/Variables de Sass
    • Sass utilities API/API de utilidades de Sass

Examples/Ejemplos

Aunque las sombras en los componentes están deshabilitadas por defecto en Bootstrap y se pueden habilitar mediante [nombre del componente $enable-shadows ], también puedes agregar o eliminar rápidamente una sombra con nuestras clases de utilidad box-shadow . Incluye compatibilidad con [nombre del componente] .shadow-none y tres tamaños predeterminados (con variables asociadas para su compatibilidad).

No shadow
Small shadow
Regular shadow
Larger shadow
HTML
 <div class="shadow-none p-3 mb-5 bg-body-tertiary rounded">No shadow</div>
 <div class="shadow-sm p-3 mb-5 bg-body-tertiary rounded">Small shadow</div>
 <div class="shadow p-3 mb-5 bg-body-tertiary rounded">Regular shadow</div>
 <div class="shadow-lg p-3 mb-5 bg-body-tertiary rounded">Larger shadow</div>

CSS

Sass maps/Mapas de Sass
scss/_variables.scss 
$box-shadow:                  0 .5rem 1rem rgba($black, .15);
$box-shadow-sm:               0 .125rem .25rem rgba($black, .075);
$box-shadow-lg:               0 1rem 3rem rgba($black, .175);
$box-shadow-inset:            inset 0 1px 2px rgba($black, .075);
Sass utilities API/API de utilidades de Sass

Las utilidades de sombra se declaran en nuestra API de utilidades en [nombre del archivo] scss/_utilities.scss. Aprenda a usar la API de utilidades.

scss/_utilities.scss 
"shadow": (
property: box-shadow,
    class: shadow,
    values: (
      null: var( --#{$prefix}box-shadow),
      sm: var( --#{$prefix}box-shadow-sm),
      lg: var( --#{$prefix}box-shadow-lg),
      none: none,
    )
  ),
