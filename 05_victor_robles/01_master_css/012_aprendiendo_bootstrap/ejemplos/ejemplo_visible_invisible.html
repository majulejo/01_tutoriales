Ejemplo de Visibilidad en Bootstrap 5

Este es un elemento visible (por defecto).
Este es un elemento invisible, pero aún ocupa espacio en el diseño.
Este es un elemento visible usando la clase d-block.
Este es un elemento completamente oculto usando la clase d-none.

Explicación del código:

  1. Elemento visible (por defecto):
    • El primer <div> con la clase alert alert-success es visible porque no tiene ninguna clase que lo oculte. Todos los elementos son visibles por defecto en HTML.
  2. Elemento invisible:
    • El segundo <div> con la clase alert alert-warning invisible está oculto visualmente, pero sigue ocupando espacio en el diseño debido a la clase invisible. Esta clase simplemente aplica visibility: hidden ; en CSS.
  3. Elemento visible con d-block:
    • El tercer <div> con la clase alert alert-info d-block es visible y usa la clase d-block para asegurarse de que el elemento tenga un display: block ;.
  4. Elemento oculto con d-none:
    • El cuarto <div> con la clase alert alert-danger d-none está completamente oculto y no ocupa espacio en el diseño. La clase d-none aplica display: none ; en CSS.

Diferencias clave entre invisible y d-none:

  • invisible : Oculta el elemento visualmente, pero mantiene su espacio en el diseño (visibility: hidden;).
  • d-none : Elimina completamente el elemento del diseño (display: none;).

Este ejemplo demuestra cómo manejar la visibilidad de los elementos en Bootstrap 5 de manera efectiva.
