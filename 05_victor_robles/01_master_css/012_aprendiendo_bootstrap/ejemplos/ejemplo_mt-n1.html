Ejemplo de uso de .mt-n1

Explicación del código:
  1. Clase .mt-n1:
    • La clase .mt-n1 aplica un margen negativo en la parte superior ( margin-top ) de -0.25rem. Esto "tira" el elemento hacia arriba, superponiéndolo ligeramente sobre el elemento anterior.
    • Es útil cuando quieres ajustar el espaciado entre elementos sin tener que modificar manualmente los estilos CSS.
  2. Estructura del ejemplo:
    • Se crean dos cuadros (div ) con la clase .box , que tienen un tamaño fijo y un fondo azul claro para visualizarlos fácilmente.
    • El primer cuadro tiene un margen inferior (mb-3 ) para separarlo del segundo cuadro.
    • El segundo cuadro usa la clase .mt-n1 para reducir el espacio entre ambos cuadros.
  3. Explicación del código:
    • Sin .mt-n1 , habría un espacio considerable entre los dos cuadros debido al margen inferior del primer cuadro.
    • Al aplicar .mt-n1 al segundo cuadro, este se mueve hacia arriba, reduciendo el espacio entre ambos.
