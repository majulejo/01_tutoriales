Explicación de cada ejemplo:

  1. position-static : Es el valor predeterminado de position. No tiene ningún efecto especial sobre el flujo del documento.
  2. position-relative : Permite posicionar elementos relativamente dentro de su contenedor. También sirve como punto de referencia para elementos con position-absolute.
  3. position-absolute : Coloca el elemento en una posición específica dentro de su contenedor relativo. En este caso, se centra usando top-50, start-50 y translate-middle.
  4. position-fixed : Fija el elemento en una posición específica de la ventana, independientemente del desplazamiento.
  5. position-sticky : Hace que el elemento se comporte como relative hasta que alcanza una posición específica (en este caso, top-0), momento en el que se "pega" a esa posición mientras se desplaza.

Notas:

  • Las cajas tienen un tamaño fijo para que puedas ver claramente cómo funcionan las utilidades de posición.
  • El ejemplo de position-fixed muestra un elemento siempre visible en la esquina inferior derecha.
  • El ejemplo de position-sticky requiere que desplaces la página para observar su comportamiento.
