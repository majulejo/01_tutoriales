Explicación del código

  1. Estilos personalizados en <style> :
    • Se definen clases CSS personalizadas para colores como indigo, púrpura, rosa, rojo y naranja.
    • Las clases para el texto siguen el patrón text-{color}.
    • Las clases para el fondo siguen el patrón bg-{color}. 
      • /* Clases para texto */
    .text-indigo {
        color: #6610f2;
    }
    .text-purple {
        color: #6f42c1;
    }
    .text-pink {
        color: #d63384;
    }
    .text-red {
        color: #dc3545;
    }
    .text-orange {
        color: #fd7e14;
    }
    
    /* Clases para fondo */
    .bg-indigo {
        background-color: #6610f2;
    }
    .bg-purple {
        background-color: #6f42c1;
    }
    .bg-pink {
        background-color: #d63384;
    }
    .bg-red {
        background-color: #dc3545;
    }
    .bg-orange {
        background-color: #fd7e14;
    }
  2. Uso de las clases personalizadas :
    • Para el texto: Se utilizan clases como text-indigo, text-purple, etc.
    • Para el fondo: Se utilizan clases como bg-indigo, bg-purple, etc.

Resultados esperados

  • Los encabezados tendrán colores de texto correspondientes a los nombres de las clases (text-indigo, text-purple, etc.).
  • Las tarjetas tendrán fondos de colores correspondientes a las clases (bg-indigo, bg-purple, etc.), y el texto será blanco para garantizar legibilidad.

Conclusión

Bootstrap 5 no proporciona clases predefinidas para todos los colores de la paleta completa. Sin embargo, puedes extender fácilmente sus funcionalidades agregando tus propias clases CSS personalizadas. Al definir estas clases, podrás aprovechar al máximo la flexibilidad de Bootstrap para aplicar cualquier color de la paleta que desees.
