Explicación de las clases de Bootstrap

A continuación, se describen las clases utilizadas en los ejemplos:

  1. .mt-0: Elimina el margen superior.
    • Se usa cuando no queremos que haya espacio encima de un elemento.
  2. .ms-1: Agrega un margen izquierdo pequeño.
    • Es útil para separar elementos horizontalmente.
  3. .px-2: Aplica relleno horizontal (izquierda y derecha).
    • Este relleno crea espacio interno a ambos lados de un elemento.
  4. .p-3: Aplica relleno uniforme en todos los lados.
    • Es ideal para dar espacio interno consistente alrededor de un contenido.
